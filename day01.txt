자바스크립트의 특징
- 웹의 동작(동적) 구현하기 위한 언어
- 웹 브라우저에서 동작(단, node.js는 서버에서 동작)
- html 문서 내에서 <script> ~ </script> 태그 안에 작성
- 대소문자를 구별
- 객체 기반의 스크립트 언어
- 웹 브라우저 자바스크립트 엔진이 내장되어 있기 때문에 사용이 가능

자바스크립트의 출력
- document.write() 함수를 통해 <body> 에 출력
- console.log() 함수를 통해 브라우저 console에 출력

외부 자바스크립트 파일로 적용
<script src = "자바스크립트 파일 경로"></script>


* 자바스크립트 파일의 확장명은 .js

자바스크립트의 주석문
한 줄 주석 : //
여러줄 주석 : /* */

프로그래밍이란?
컴퓨터의 프로그램을 작성하는 말, 일반적으로 프로그램 작성 방법의 결정, 코딩, 에러 수정등의 작업을 모두 가리킨다.

변수(Variable)
변수란 데이터를 저장할 수 있는 메모리 공간을 의미한다.
값이 변경될 수 있으며 자바스크립트에서는 let 키워드를 사용하여 변수를 선언하고 선언되지 않은 변수를 사용하거나 접근하려고 하면 오류가 발생한다.

변수명 선언시 주의점
* 예약어(자바스크립트 명령어) 사용 금지
* 공백은 허용하지 않는다.
* 숫자로 시작할 수 없다.
* 사용 가능한 문자는 영문자, 숫자, _, $ 사용 가능

var를 사용하지 않는 이유
1. 같은 이름의 변수를 선언할 수 있다.
2. 지역변수와 전역변수의 구별이 없다.

지역변수(Local variable) : 특정구역({ }) 내에서 생성되어 그 구역에서만 사용
(함수 속에 선언되어 해당 함수 속에서만 사용이 가능한 변수)
전역변수(Global variable) : 어느 위치에서든 호출하면 사용이 가능
(함수 밖에 선언하여 클래스 전체에서 사용이 가능한 변수)

use strict
strict mode 사용을 한다는 의미, 코드를 엄격하게 사용하도록 한다.

상수(constant)
한 번 선언한 상수는 다시 재정의 할 수 없으며, 값도 재할당 할 수 없다.

const 상수명;
상수명 = 값 (x)

const 상수명 = 값; (o)
상수명 = 값(x) // 재할당 할 수 없다.

typeof() 함수
타입을 나타내는 문자열을 리턴한다.


자바스크립트의 데이터 타입(형)
타입이란 프로그램에서 다룰 수 있는 값의 종류를 의미한다.

1. 숫자형(number)
자바스크립트는 다른 언어와 달리 정수와 실수를 따로 구분하지 않는다. 모든 수를 실수 하나로만 표현한다.

2. 문자열형(string)
자바스크립트는 문자열을 쌍따옴표, 따옴표로 둘러싸인 문자의 집합을 의미한다.
